## 欢迎来到我的小窝

这是在这个博客里落下的第一行字。就像刚搬进新家，总想先在每个角落摸摸看，看看灯亮不亮，窗外的风景好不好。

在这儿，我想试试把那些冷冰冰的技术，揉进一点点生活的温度。

### 当 Python 遇上问候

写代码的时候，我总觉得它们是有灵气的。比如这几行 Python，简单得像是在晨间给远方的朋友寄出一张明信片：

```python
def hello_world():
    # 这一句，是我想对这个世界说的悄悄话
    print("你好呀，华江的小世界！")

if __name__ == "__main__":
    hello_world()
```
