# 需求说明书 (Requirements)

## 1. 当前已实现功能 (As-Is)

### 1.1 内容管理
- [x] **Markdown/MDX 支持**: 支持 GFM, 数学公式 (KaTeX), 代码高亮 (Prism), 引用。
- [x] **标签系统 (Tags)**: 文章分类、标签云、标签筛选页。
- [x] **作者多角色**: 支持多作者配置 (Authors)。

### 1.2 用户交互
- [x] **评论系统**: 集成 Giscus (基于 GitHub Discussions)。
- [x] **全文搜索**: 集成 kbar (命令面板风格)。
- [x] **暗黑模式**: 随系统自动切换或手动切换。

### 1.3 页面功能
- [x] **首页**: 展示最新文章、个人简介摘要。
- [x] **博客列表**: 分页、标签筛选。
- [x] **项目页 (Projects)**: 展示个人作品集 (`data/projectsData.ts`)。
- [x] **关于页 (About)**: 个人详细介绍。

### 1.4 技术/非功能需求
- [x] **SEO**: 自动生成 Sitemap, Robots.txt, Meta 标签。
- [x] **分析**: 集成 Umami / Plausible (可选)。
- [x] **RSS**: 自动生成 RSS 订阅源。

## 2. 待开发需求 (To-Be)
> 在此处添加新需求，AI 将读取此部分以规划工作。

- [x] **多语言支持 (i18n)**:
  - [x] 实现类似 `nelsonlai.dev` 的语言切换功能。
  - [x] **路由设计**: 引入 `[locale]` 动态路由前缀（如 `/zh-CN/blog`, `/en/blog`）。
  - [x] **内容翻译**: 支持 MDX 文章的多语言版本（通过 Frontmatter 字段或文件夹区分）。
  - [x] **UI 国际化**: 导航栏、页脚及系统按钮支持多语言字典切换。
  - [x] **语言检测**: 中间件自动检测浏览器语言并重定向至默认或匹配语言。
